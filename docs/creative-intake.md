# Creative Intake

Human Chapter Intake 用于接住作者亲写正文或正文片段，不润色、不改写、不直接入正史。

命令：

```bash
novel intake chapter <projectName> <inputId>
```

生成 Creative Intake Capsule：

```text
fact_delta.yaml
atmosphere_triads.md
confirmed_vibes.md
tentative_vibes.md
intention_hypotheses.yaml
conflict_footnotes.md
retcon_debt_update.yaml
style_candidates.md
memory_patch.yaml
alignment_questions.md
```

原则：

- fact 可以提取，但仍要保留来源。
- atmosphere 必须三角验证。
- intention 必须分 L1/L2/L3，并带 evidence、confidence、ttl。
- L3 弱猜测不得进入 decision log。
- memory patch 需要作者确认。
