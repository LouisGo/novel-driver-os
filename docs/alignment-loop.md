# Alignment Loop

Weekly Alignment 用于校准系统是否误解作者，而不是生成冗长报表。

命令：

```bash
novel align weekly <projectName>
```

报告包含：

1. 本周输入回顾。
2. 系统不确定的问题。
3. 需要作者确认的关键项。
4. 风格偏移观察。
5. 下周建议。

来源：

- Author Input Packets
- `intention_hypotheses.yaml`
- `tentative_vibes.md`
- `open_questions.md`
- `style_candidates.md`
- `anti_style.md`

报告不修改正史，只为作者确认和系统策略调整提供依据。
