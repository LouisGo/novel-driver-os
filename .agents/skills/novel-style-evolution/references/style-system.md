# 文风系统参考

用于 `novel-style-evolution`。

## 文件职责

- `style_bible.md`：只放作者确认的稳定规则。
- `aspirational_style.md`：作者想靠近的方向，不等于模仿。
- `anti_style.md`：明确不要的表达、节奏、桥段、AI 味。
- `discarded_brilliance.md`：废案和未来复活资产。
- `style_entropy_budget.md`：保留实验空间，防止文风锁死。

## 升级规则

- 文风候选 不能直接进入 style_bible。
- 至少需要作者确认或多次高权威输入支持。
- 每条规则都要有“适用范围”和“过度使用风险”。

## 中文网文维度

- 追读节奏。
- 爽点密度。
- 章末钩子强度。
- 对白信息量。
- AI 味风险。
- 关系线表达方式。
